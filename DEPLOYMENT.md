# Deployment Guide

## Quick Start

1. **Add Your Content**:
   - Place your resume PDF at `/resume.pdf`
   - Add project images to `/images/[project-name]/`
   - Add project reports to `/reports/`
   - Update contact info in `index.html`

2. **Test Locally**:
   - Open `index.html` in a web browser
   - Test all navigation and features
   - Verify images and PDFs load correctly

3. **Deploy**:
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Your site will be live at `https://username.github.io/repository-name`
